### `reactit` -  **React**ion **It**erator

`reactit` is a code for generating all possible reactions between species in an efficient manner.


#### Installation 

```
git clone https://github.com/badw/reactit.git 
cd reactit 
pip install . 
```

#### Examples 

`./examples/reactit.ipynb`
